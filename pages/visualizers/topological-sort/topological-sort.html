4. u = Queue.dequeue(), add to Topo Order
5. For each neighbor v of u:
6. Remove edge u → v, InDegree[v]--
7. If InDegree[v] == 0: Queue.enqueue(v)
8. If Topo Order size == V, return Topo Order

Theory

Kahn's Algorithm finds a valid topological ordering of a Directed Acyclic Graph (DAG) by stripping away dependencies starting from nodes that require nothing.

  • In-Degree The number of prerequisites a node relies on.
  • 0-Queue Nodes that have no remaining dependencies.

Complexity

Time Complexity O(V + E)
Space Complexity O(V)

Every vertex (V) and edge (E) is evaluated exactly once, making it strictly linear.
