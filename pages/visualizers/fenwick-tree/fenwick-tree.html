Theory

A Fenwick Tree computes prefix sums and updates elements in logarithmic time. Instead of a standard tree structure with left/right pointers, it cleverly uses the binary representation of array indices.

  • LSB (i & -i) The Least Significant Bit determines the size of the range a node is responsible for.
  • Update Jump Moving forward by adding the LSB `i += LSB` cascades the update to all parent intervals.
  • Query Jump Moving backward by subtracting the LSB `i -= LSB` aggregates non-overlapping sub-ranges.
