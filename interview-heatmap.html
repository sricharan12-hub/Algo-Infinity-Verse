Problem Statement

Longest Increasing Subsequence

Given an integer array nums, return the length of the longest strictly increasing subsequence. A subsequence is a sequence derived from the array by deleting some or no elements without changing the order of the remaining elements.

You must design an algorithm that runs in better than O(n²) time complexity if possible, though an O(n²) dynamic programming solution is also acceptable for a first pass.

Constraints
  • 1 <= nums.length <= 2500
  • -10⁴ <= nums[i] <= 10⁴
  • All elements may include negative numbers and duplicates.
  • The subsequence does not need to be contiguous.
Examples
Example 1: 
Input: nums = [10,9,2,5,3,7,101,18]
Output: 4
Explanation: The longest increasing subsequence is [2,3,7,101], length 4.
Example 2: 
Input: nums = [0,1,0,3,2,3]
Output: 4
